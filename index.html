    ASCII Code - The Extended ASCII Table

    ASCII abbreviated from American Standard Code for Information Interchange, is a character encoding standard for electronic communication. ASCII codes represent text in computers, telecommunications equipment, and other devices. Most modern character-encoding schemes are based on ASCII, although they support many additional characters. The ASCII characters can be divided into several groups.

    • ASCII control characters
    • ASCII printable characters
    • The extended ASCII codes

    ASCII control characters (character code 0-31)

    The first 32 characters in the ASCII-table are unprintable control codes and are used to control peripherals such as printers.

    ASCII printable characters (character code 32-127)
